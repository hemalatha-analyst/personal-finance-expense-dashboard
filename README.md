# 💰 Personal Finance & Expense Management Dashboard

## 📊 Objective

To build an Excel-based dashboard that tracks income, expenses, savings, and financial performance using KPIs and visual analytics.

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts (Line, Bar, Donut)
* Slicers & Dashboard Design

---

## 📁 Dataset

The dataset includes:

* Date, Month
* Income Source
* Expense Category
* Payment Mode
* Amount
* Indicator (Income/Expense)

---

## 📌 Features

* KPI Cards (Income, Expense, Savings, Savings %)
* Monthly Income vs Expense Trend
* Expense Distribution Analysis
* Payment Mode Insights
* Interactive Filters (Slicers)
